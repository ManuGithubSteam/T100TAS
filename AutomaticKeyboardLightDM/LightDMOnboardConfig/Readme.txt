
This files will make the onboard keybord in lightdm (unity greeter) smaller so it will fit in portrait mode. Also they set a wallpaper from KDE but it can be any picture.

Files belong into:

/usr/share/glib-2.0/schemas/
/usr/share/onboard/


After placeing the files execute:

glib-compile-schemas /usr/share/glib-2.0/schemas/
